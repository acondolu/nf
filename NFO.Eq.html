"Aczel" equality:

Definition Aeq {X Y} f g :=
  (∀ x: X, ∃ y, f x == g y) ∧ ∀ y: Y, ∃ x, f x == g y.


eeq is an equivalence relation:
Lemma eeq_refl: ∀ x, eeq x x.

Lemma eeq_sym: ∀ x y, eeq x y → eeq y x.

Lemma eeq_trans : ∀ x y z, eeq x y → eeq y z → eeq x z.

Instance nfo_setoid : Equivalence eeq.

Aeq is an equivalence
Lemma Aeq_refl: ∀ {X} f, @Aeq X X f f.

Lemma Aeq_sym: ∀ {X Y} f g, @Aeq X Y f g → Aeq g f.

Lemma Aeq_trans: ∀ {X Y Z} f g h, @Aeq X Y f g → @Aeq Y Z g h → Aeq f h.

The good unfolding lemma for eeq:
Lemma eeq_unfold {X' Y' f' g' e' X Y f g e}:
  eeq (S X Y f g e) (S X' Y' f' g' e')
    ↔ Aeq f f' ∧ Qeq (map g e) (map g' e').
