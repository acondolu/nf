Library NFO.Xor



This module defines exclusive disjunction (xor), noted by ⊻. Note: xor is a fundamentally classical operator, and in fact from now on we freely use classical logic.

Definition xor p q := (p ∧ ¬q) ∨ (¬p ∧ q).
Infix "⊻" := xor (at level 80, right associativity).

A bunch of trivial but useful properties about xor follow.
Self-inverse:
Lemma xor_absorb : ∀ p, p ⊻ p ↔ False.

Commutativity:
Lemma xor_comm: ∀ p q, p ⊻ q ↔ q ⊻ p.

Associativity:
Lemma xor_assoc: ∀ p q r, p ⊻ (q ⊻ r) ↔ (p ⊻ q) ⊻ r.

Left and right identity:
Lemma xor_false_l: ∀ p, False ⊻ p ↔ p.

Lemma xor_false_r: ∀ p, p ⊻ False ↔ p.

xor is a morphism:
Lemma xor_iff: ∀ p p' q q', (p ↔ p') → (q ↔ q') → p ⊻ q ↔ p' ⊻ q'.
Add Morphism xor: eeqs.

Negation of xor:
Lemma xor_neg: ∀ p q, ¬ (p ⊻ q) ↔ (p ↔ q).

Projections:
Lemma xor_1: ∀ p q: Prop, p → ¬q → p ⊻ q.

Lemma xor_2: ∀ p q: Prop, ¬p → q → p ⊻ q.

Xor commutes with negation:
Lemma xor_neg_commute: ∀ p q, p ⊻ ¬q ↔ ~(p ⊻ q).

A technical lemma that interleaves the operands:
Lemma xor_pairs: ∀ p q r s, (p ⊻ q) ⊻ (r ⊻ s) ↔ (p ⊻ r) ⊻ (q ⊻ s).

Exclusive disjunction of predicates:
Definition xorP {X} (P Q: X → Prop) := fun x ⇒ P x ⊻ Q x.
Infix "^^" := xorP (at level 80, right associativity).

If P and Q are R-morphisms, then P ^^ Q is an R-morphism as well:
Lemma xorP_respects {X} (R: X → X → Prop) (P Q: X → Prop) :
  respects R P → respects R Q → respects R (P ^^ Q).
