Library Internal.WfMult: Well-founded multiset extension

This module defines the multiset extension of a well-founded binary relation.
We represent multisets simply as lists. Then we use permutations so to ignore the position of elements in a list.

Section WfMult.

We assume a well-founded relation << over A:
Variable A : Type.
Variable lt : A → A → Prop.
Variable wf_lt: well_founded lt.

Replace the element of a list at a certain position by concatenating another list in its place:
Definition replace : ∀ {i: nat} {l: list A},
  i < length l → list A → list A.

Lemma replace_cons: ∀ {i x l l'} (p: S i < length (x::l)) (p': i < length l),
  replace p l' = x :: replace p' l'.

Get an element of the list by its index:
Definition get : ∀ {i: nat} {l: list A}, i < length l → A.

Lemma get_cons: ∀ {i x l} (p: S i < length (x::l)) (p': i < length l),
  get p = get p'.

Check whether a predicate is satisfied by all the elements of a list:
Fixpoint all P (l: list A) := match l with
| nil ⇒ True
| cons b bs ⇒ P b ∧ all P bs
end.

The extension of << to lists. l <<< l' holds iff there exists an element x in l such that l' is obtained from l by replacing that element with a (possibly empty) list whose elements are all strictly smaller than x.
Inductive ltl : list A → list A → Prop :=
  | C : ∀ i (l l': list A) (p: i < length l),
         all (fun x ⇒ x << get p) l'
          → ltl (replace p l') l .

Comment:

Lemma ltl_cons: ∀ b a a',
  ltl a a' → ltl (b :: a) (b :: a').

Lemma ltl_concat_left: ∀ a b b',
  ltl b b' → ltl (a ++ b) (a ++ b').

Lemma ltl_concat_right: ∀ b a a',
  ltl a a' → ltl (a ++ b) (a' ++ b).

A useful inversion lemma:
Lemma ltl_cases: ∀ {a l l'},
  l <<< a :: l'
    → (∃ l'', l'' <<< l' ∧ l = a :: l'')
      ∨ ∃ l'', l = l'' ++ l' ∧ all (fun x ⇒ lt x a) l''.

We follow the proof by Tobias Nipkov in "An inductive Proof of the Wellfoundedness of the Multiset Order". We adapt the lemmas in that document to the case of lists instead of multisets.
Note: the definition of <<< above is justified by our need to keep the proofs nicely inductive on the structure of lists. We are forced to concatenate the new list of smaller elements exactly at the position where the removed element was.
Lemma 2.1:
Lemma l2p1: ∀ {a M0},
  (∀ b, b << a → ∀ M, Acc ltl M → Acc ltl (b :: M))
    → Acc ltl M0
      → (∀ M, M <<< M0 → Acc ltl (a :: M))
        → Acc ltl (a :: M0).

Lemma 2.2:
Lemma l2p2: ∀ a,
  (∀ b, b << a → ∀ M, Acc ltl M → Acc ltl (b :: M))
    → ∀ M, Acc ltl M → Acc ltl (a :: M).

Lemma 2.3:
Lemma l2p3: ∀ {a} M, Acc ltl M → Acc ltl (a :: M).

Theorem wf_ltl: well_founded ltl.

Unordered

To consider lists as multisets, we now use permutations so to make our order relation ignore the position of elements in a list.
We use Coq's Permutation library (see https://coq.github.io/doc/master/stdlib/Coq.Sorting.Permutation.html).

Definition ltlp l l' := ∃ l'', Permutation l l'' ∧ l'' <<< l'.

Lemma perm_lt_dx: ∀ {a b b'}, Permutation b b' → a <<< b' → ltlp a b.

Lemma perm_Acc : ∀ l l', Permutation l l' → Acc ltlp l → Acc ltlp l'.

Theorem wf_perm : well_founded ltlp.


Definition lltlp := clos_trans _ ltlp.
Definition wf_trans: well_founded lltlp.

Aux: comment
Lemma lltlp_concat_left: ∀ a b b',
  lltlp b b' → lltlp (a ++ b) (a ++ b').

Lemma lltlp_concat_right: ∀ b a a',
  lltlp a a' → lltlp (a ++ b) (a' ++ b).

Lemma lltlp_concat: ∀ a a' b b',
  lltlp a a' → lltlp b b' → lltlp (a ++ b) (a' ++ b').

Lemma l_perm_lt_sx: ∀ {a a' b},
  Permutation a' a → lltlp a b → lltlp a' b.
