Section WfDecr.

Variable A : Type.
Variable lt : A → A → Prop.
Variable wf_lt: well_founded lt.

We define the decr binary relation between lists, such that decr l l' iff every element of l is smaller than some element of l'.
Definition decr' l l': Type :=
  prod (l' ≠ nil) (allT (fun a ⇒ someT (lt a) l') l).

gather l l' a p gathers all the elements of l that are smaller than the first element of l'.
Definition gather: ∀ l l' a',
  allT (fun a ⇒ someT (lt a) (a' :: l')) l → list A.
Defined.

Correctness of gather:
Lemma gather_ok: ∀ {l l' a'},
  ∀ (p: allT (fun a ⇒ someT (lt a) (a' :: l')) l),
    allT (fun a ⇒ lt a a') (gather _ _ _ p).

drop l l' a p drops all the elements from l that are smaller than the first element of l'. (It is the dual of gather).
Definition drop: ∀ l l' a',
  allT (fun a ⇒ someT (lt a) (a' :: l')) l → list A.
Defined.

Correctness sof drop:
Lemma drop_ok: ∀ {l l' a'},
  ∀ (p: allT (fun a ⇒ someT (lt a) (a' :: l')) l),
    allT (fun a ⇒ someT (lt a) l') (drop _ _ _ p).

We use gather and drop to permutate a list l, moving all the gathered elements to the head of the list:
Lemma gather_drop_ok: ∀ l l' a a0,
 Permutation l (gather l l' a a0 ++ drop l l' a a0).

An important result. We show that decr is contained in the transitive closure of the permutation order between lists defined in Internal.WfMult.
To prove it, we proceed by induction on l', using gather_drop_ok and concluding by i.h.
Theorem decr'_ok: ∀ l l', decr' l l' → lltlp lt l l'.

Definition decr (a b: list A) : Prop := ☐ (decr' a b).

As a consequence, we easily obtain well-foundedness of decr by the inclusion theorem.
Theorem wf_decr: well_founded decr.

Lemma decr_unfold : ∀ l l',
  decr l l'
    ↔ l' ≠ nil ∧ all (fun a ⇒ some (lt a) l') l.

A nicer unfolding lemma for decr, using In instead of all and some. We won't use this version, thought.
Lemma decr_unfold_2 : ∀ l l',
  decr l l'
    ↔ l' ≠ nil ∧ ∀ x, In x l → ∃ y, In y l' ∧ lt x y.
